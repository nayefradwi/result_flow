Result

Result<T> class abstract

A class that represents the result of an operation. It can be either a success using ResultWithData or an error using ResultWithError.

The abstract class Result provides a way to handle the result of one or more operations in a safe and functional way by providing methods like on, onAsync, mapTo, continueWith, and handle.
